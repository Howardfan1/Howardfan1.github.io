# Markdown 写作指南

Markdown 是一种轻量级标记语言，让你可以用简单的纯文本格式来写出排版优美的文章。

<!-- more -->

## 基本语法

### 标题

使用 `#` 号来创建不同级别的标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本格式

| 格式 | 语法 | 效果 |
|------|------|------|
| 加粗 | `**加粗文本**` | **加粗文本** |
| 斜体 | `*斜体文本*` | *斜体文本* |
| 删除线 | `~~删除线~~` | ~~删除线~~ |
| 行内代码 | `` `code` `` | `code` |

### 列表

**无序列表：**

- 项目一
- 项目二
  - 子项目 A
  - 子项目 B
- 项目三

**有序列表：**

1. 第一步
2. 第二步
3. 第三步

### 引用

> 学而不思则罔，思而不学则殆。
>
> — 《论语》

### 代码块

支持语法高亮：

```python
def hello(name: str) -> str:
    """打个招呼"""
    return f"Hello, {name}!"

print(hello("World"))
```

```javascript
const greet = (name) => `Hello, ${name}!`
console.log(greet('World'))
```

### 链接和图片

```markdown
[链接文本](https://example.com)
![图片描述](https://example.com/image.jpg)
```

## 写作建议

1. **标题要清晰** — 让读者一眼就知道文章内容
2. **段落要短** — 避免大段文字，保持可读性
3. **善用列表** — 条理清晰地组织信息
4. **代码要高亮** — 指定语言获得语法高亮
5. **配图说话** — 一图胜千言
